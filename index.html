Get the Extension

Install the Chrome extension to trace concepts directly from any webpage. Simply select text and press Ctrl+Shift+G (or Cmd+Shift+G on Mac).

Download Extension v1.0
Installation Instructions
  1. Download the extension file above
  2. Open Chrome and go to chrome://extensions/
  3. Enable "Developer mode" in the top right
  4. Drag and drop the downloaded file onto the page
  5. Select text on any webpage and press Ctrl+Shift+G
